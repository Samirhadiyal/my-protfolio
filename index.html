Smart Budget Tracker - Streamlit-Based App to Track Monthly Expenses

This project is a user-friendly web application built using Streamlit that helps individuals track and analyze their monthly income and expenses. Users can input their income and categorize their expenses (e.g., food, transport, utilities, etc.) through a simple interface. The app processes this data and provides visual insights in the form of bar charts, pie charts, and summary tables, allowing users to better understand their spending habits and manage their budget effectively. The application leverages Python libraries such as Pandas for data manipulation, NumPy for numerical operations, Matplotlib for data visualization, and Streamlit for building the interactive web interface.

Virtual White Board

Virtual White Board is a Java Swing-based drawing application that allows users to sketch freely on a digital canvas using mouse input. Designed as a beginner-friendly project, it demonstrates the fundamentals of Java GUI development, including real-time rendering, mouse event handling, and custom graphics using the Graphics class. The application features a clean and minimal interface, built entirely with core Java libraries and without any external dependencies, making it easy to understand, run, and modify. Ideal for students and aspiring developers, this project serves as a practical introduction to interactive desktop application development and lays the groundwork for enhancements like color selection, brush customization, and canvas saving.

Galactic Defender

Galactic Defender is a fast-paced 2D space shooter game built using Python and Pygame, where players control a spaceship to defend the galaxy from waves of incoming alien invaders. The player must move left and right to dodge enemies while firing bullets to destroy them before they reach Earth. Featuring background music, sound effects, score tracking, and multiple enemies with dynamic movements, the game offers an exciting arcade-style experience that tests reflexes and accuracy. It's a great demonstration of real-time game logic, event handling, and collision detection in Python.
